IntroductionÂ¶

QXMD is a Quantum Molecular Dynamics (QMD) simulation software with various eXtensions. QMD follows the trajectories of all atoms while computing interatomic forces quantum mechanically in the framework of density functional theory (DFT). [RN114] The QXMD software has been developed by Fuyuki Shimojo since 1994. [RN334] Since 1999, various extensions have been developed in collaboration with Rajiv Kalia, Aiichiro Nakano and Priya Vashishta. [RN182] The basic QXMD code is based on a plane-wave basis to represent electronic wave functions and pseudopotential (PP) methods to describe electron-ion interaction. Supported PPs include norm-conserving PP [RN116] and ultrasoft PP [RN211] , as well as an all-electron projector augmented-wave (PAW) method [RN263]. Electron-electron interaction beyond the mean-field Hartree approximation is included using various exchange-correlation functionals, with and without spin polarization: generalized gradient approximation (GGA) [RN277], DFT+U method for transition metals [RN322], van der Waals (vDW) functional for molecular crystals and layered materials [RN21], nonlocal correlation functional [RN126], and range-separated exact-exchange functional [RN324] [RN99]. Various unique capabilities included in the QXMD code (some of which are described in [RN181]) include:

    1. Linear-scaling DFT algorithms [RN181] [RN184] [RN128]
    2. Scalable algorithms on massively parallel computers [RN139] [RN139] [RN64]
    3. Nonadiabatic quantum molecular dynamics (NAQMD) to describe excitation dynamics [RN329]
    4. Omni-directional multiscale shock technique (OD-MSST) to study shock response of materials [RN250] [RN251]
