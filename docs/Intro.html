Introduction¶

QXMD is a Quantum Molecular Dynamics (QMD) simulation software with various eXtensions. QMD follows the trajectories of all atoms while computing interatomic forces quantum mechanically in the framework of density functional theory (DFT). [1] The QXMD software has been developed by Fuyuki Shimojo since 1994. [2] Since 1999, various extensions have been developed in collaboration with Rajiv Kalia, Aiichiro Nakano and Priya Vashishta. [3] The basic QXMD code is based on a plane-wave basis to represent electronic wave functions and pseudopotential (PP) methods to describe electron-ion interaction. Supported PPs include norm-conserving PP [4] and ultrasoft PP [5] , as well as an all-electron projector augmented-wave (PAW) method [6]. Electron-electron interaction beyond the mean-field Hartree approximation is included using various exchange-correlation functionals, with and without spin polarization: generalized gradient approximation (GGA) [7], DFT+U method for transition metals [8], van der Waals (vDW) functional for molecular crystals and layered materials [9], nonlocal correlation functional [10], and range-separated exact-exchange functional [11] [12]. Various unique capabilities included in the QXMD code (some of which are described in [13]) include:

    1. Linear-scaling DFT algorithms [13] [14] [15]
    2. Scalable algorithms on massively parallel computers [16] [16] [17]
    3. Nonadiabatic quantum molecular dynamics (NAQMD) to describe excitation dynamics [18]
    4. Omni-directional multiscale shock technique (OD-MSST) to study shock response of materials [19] [20]
